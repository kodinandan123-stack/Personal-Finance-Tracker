# Roadmap

Planned features and improvements for the Personal Finance Tracker.

## v1.0 (MVP) — ✅ Complete
- [x] Income and expense tracking by category
- [x] User authentication (JWT)
- [x] Monthly dashboard with charts

## v1.1 — ✅ Complete
- [x] Savings goals tracker
- [x] CSV export of transactions
- [x] Recurring transactions

## v2.0 — ✅ Complete
- [x] Budget alerts and notifications
- [x] Multi-currency support
- [x] Mobile app (Expo/React Native client) — see MOBILE.md

## v2.1 — ✅ Complete
- [x] Structured health check endpoint with DB status
- [x] Password reset via email flow
- [x] Extended validator helpers

## v2.2 — ✅ Complete
- [x] Recurring transaction routes (CRUD, pause, resume, process-due)
- [x] Email helper (budget alert, goal milestone, welcome, monthly summary)
- [x] Tax calculator (US income, capital gains, self-employment, quarterly estimates)

## v2.3 — ✅ Complete
- [x] Analytics routes (spending breakdown, income-vs-expense, net worth)
- [x] Dark mode preference persistence
- [x] Transaction search and filter by date range, category, and amount

## v3.0 — ✅ Complete (Mobile)
- [x] Navigation setup (React Navigation + tab navigator)
- [x] Login and Register screens
- [x] Dashboard screen with charts
- [x] Transactions screen
- [x] Budgets screen
- [x] Goals screen
- [x] Recurring transactions screen
- [x] Reports screen
- [x] Profile screen
- [x] Shared components (notification banner, currency helper, debounce)
- [x] App entry point (App.js) and Babel config wired up for Expo

## Future Ideas
- [ ] Shared/household accounts
- [ ] Bank account sync (Plaid integration)
- [ ] Automated test coverage (backend + frontend)
